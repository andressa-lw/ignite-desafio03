## 🚀 Como executar o projeto

### Pré-requisitos

Antes de começar, você vai precisar ter instalado em sua máquina as seguintes ferramentas:
[Git](https://git-scm.com), [Node.js](https://nodejs.org/en/). 

#### 🧭 Rodando o server

```bash

# Clone este repositório
$ git clone https://github.com/rocketseat-education/ignite-template-reactjs-criando-um-hook-de-carrinho-de-compras

# Acesse a pasta do projeto no seu terminal/cmd
$ cd nome-da-pasta

# Instale as dependências
$ yarn # ou npm install

# Execute o server
$ yarn server # ou npm run server

# O server estará rodando na porta:3333 - acesse http://localhost:3333

```

#### 🧭 Rodando a aplicação

Com o repositório clonado anteriormente, é preciso somente executar a aplicação agora.

```bash

# Execute a aplicação em modo de desenvolvimento
$ yarn dev # ou npm run dev

# A aplicação será aberta na porta:3000 - acesse http://localhost:3000

```
